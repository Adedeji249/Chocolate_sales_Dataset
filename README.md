# 🍫 Chocolate Sales Analysis
This project presents an end-to-end data analysis of a chocolate sales dataset using SQL, Excel, and Power BI. It focuses on cleaning raw data, analyzing key business metrics, and building an interactive dashboard to generate actionable insights.

## 🎯 Objectives
- Analyze revenue and profit performance  
- Identify top-performing products and regions  
- Understand customer behavior and segmentation  
- Provide insights for data-driven decision-making

## 🛠️ Tools Used
- SQL (Data querying and transformation)  
- Microsoft Excel (Data cleaning and pivot tables)  
- Power BI (Dashboard and visualization)

## 📊 Dataset
The dataset includes:
- Customer ID  
- Product Name  
- Revenue, Cost, Profit  
- Order Date  
- Customer demographics (Age, Gender)  
- Location (City, Country)  
- Store Type

## 🔍 Process
1. Data Integration (SQL)
    - Used SQL to combine multiple datasets from different sources into a single, structured            dataset.
    - Performed joins across tables to create a unified view for analysis.
2. Data Cleaning (Excel)
   - Removed duplicates and handled missing values  
   - Standardized data formats
   - Remove columns that are not needed
3. Data Visualization (Power BI)
   - Built interactive dashboard  
   - Created KPIs and charts for business insights
  
## 📈 Key Insights
- Dark chocolate products generated the highest revenue and profit 
- Customers aged 25–44 contributed the most sales  
- Airports were the top-performing sales store  
- Canada showed high revenue and while UK showed highest profit margins
